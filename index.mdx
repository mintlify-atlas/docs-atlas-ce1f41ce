<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your portfolio up and running in minutes</p>
  
  <Steps>
    <Step title="Clone and install dependencies">
      Clone the repository and install all required packages using your preferred package manager.
      
      ```bash
      git clone https://github.com/mrcl29/jowy-portfolio.git
      cd jowy-portfolio
      pnpm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Set up your API credentials for Spotify, SoundCloud, and YouTube integrations.
      
      ```bash
      cp .env.example .env
      # Edit .env with your API keys
      ```
      
      <Note>You'll need API credentials from Spotify, SoundCloud, and YouTube. See the [API Setup guide](/guides/api-setup) for details.</Note>
    </Step>
    
    <Step title="Start the development server">
      Launch the Astro development server and view your portfolio locally.
      
      ```bash
      pnpm dev
      ```
      
      Visit `http://localhost:4321` to see your portfolio in action.
    </Step>
  </Steps>
</div>
